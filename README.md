# 🧠 IndegenousMCU32

**IndegenousMCU32** is a 32-bit RISC-V based Microcontroller Unit (MCU) designed for embedded and real-time applications.

---

## 🔹 Core Architecture

- **ISA:** RV32IMAC  
  - I – Integer Base Instruction Set  
  - M – Integer Multiplication & Division  
  - A – Atomic Instructions  
  - C – Compressed Instructions  

- **Architecture:** 32-bit RISC-V
- **Operating Frequency:** 75 – 100 MHz

---

## 🔹 On-Chip Peripherals

### ⚙️ PWM (Pulse Width Modulation)
- 6 PWM Channels

### 🔌 GPIO
- 16 General Purpose Input/Output Pins

### 🔄 SPI (Serial Peripheral Interface)
- 3 SPI Interfaces

### 🔁 I2C (Inter-Integrated Circuit)
- 2 I2C Controllers

### 💾 External Memory Support
- Supports up to 512 Mb SDRAM

### 📡 UART (Universal Asynchronous Receiver Transmitter)
- 3 UART Interfaces

### 🧪 Debug Interface
- JTAG Support for Programming & Debugging
